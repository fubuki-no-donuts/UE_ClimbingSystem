# ARPG System


## 项目描述

本项目旨在实现一个 ARPG 类型的游戏模板，实现游戏中的大部分底层框架，包括但不限于：动作系统、任务系统、背包系统、数值系统、UI 系统等

## 项目进度

本项目预计实现的系统，

- [ ] 运动系统
  - [x] 攀爬系统
  - [ ] 战斗系统
- [ ] 任务系统
- [ ] 背包系统
- [ ] 数值系统
  - [ ] 伤害系统
  - [ ] 成长系统
- [ ] UI 系统
  - [ ] 红点系统
- [ ] 事件系统
